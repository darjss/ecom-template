# Contract authority index

Use this index before changing a feature. It points to contract owners; it does not restate their bodies.

## Precedence

1. The [final scope reconciliation](../wayfinder/final-scope-reconciliation.md) wins wherever it names a conflict or simplification.
2. The accepted feature contract governs the remaining details for its feature.
3. Research, prototypes, the [landed bootstrap plan](../architecture/bootstrap-plan.md), and other historical artifacts provide evidence or implementation navigation only. They do not override an accepted contract.

[Issue #33](https://github.com/darjss/ecom-template/issues/33) is the current implementation scope synthesized from these decisions. If two accepted contracts still conflict after applying the final reconciliation, stop and report the exact clauses rather than choosing silently.
